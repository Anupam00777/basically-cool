The Calculator

Everyone makes it, so I made one too. It's a simple Calculator but with some additional options like dark mode and a notebook that will save all the calculations as history in local storage of device.

A 3D world with a moving character

This was a project that took lots of learning and understanding of how 3d works. From matrices to vectors, it was a long run. The 3D engine used here is three.js which simulates a 3d world in browser using javascript. Also to add physics to the world I used cannon-es, a physics engine written in JS. Other libraries I used are Three-to-cannon and hammer.js.

A PONG game using Javascript

I used pure HTML, CSS and JAVASCRIPT to make this game of pong. Its not made on html canvas, instead i simulated it using gtml divs and js. So its now working perfectly.

Portfolio Website made by ChatGPT

I asked ChatGPT to make a portfolio website for showcasing a web developers projects, and then to make it look visually good. I then asked it to make some little changes and to make some components more appealing. Considering how much time it took and I just had to prompt 7 times, the result is pretty good. It's not the most beautiful and functional of a website, but it was indeed made in just under 5 min, with practically 0 human intervention. I just changed some default texts and thats all.

Typing Practice

This is a Typing Practice web application made with HTML, Tailwind CSS and pure Javascript. Tailwind CSS is a framework of CSS which makes writing responsive CSS super Easy. I truly like how this turned out. Other than a fun Application, it is really helpful too. The dataset for words is from Kaggle about top most common english words.
